# Flutter 通用模板项目

## 开始

```
git clone https://github.com/keno-lee/flutter-template.git

flutter run
```

## 目录结构

```
|____ .idea - 编辑器通用配置
|____ android 安卓配置
|____ build 打包文件
|____ ios IOS配置
|____ lib 开发目标文件夹
| |____ api http请求文件
| |____ assets 静态资源
| |____ components 公共组件
| |____ model 数据模型
| |____ routers 路由模块
| |____ style 样式组件
| |____ utils 工具包
| | |____ format 格式化金额
| | |____ http Http请求
| |____ views 页面
| |____ main.dart 入口文件（相当于vue中main.js)
|____ plugins
|____ test
|____ gitignore
|____ .metadata
|____ .packages
|____ pubspec.lock
|____ pubspec.yaml
|____ README.md
|____ template.iml
```
